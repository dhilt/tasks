	Асинхронное программирование. Promises

1. Познакомиться с ES6 Promises
 - http://exploringjs.com/es6/ch_promises.html
 - http://www.datchley.name/es6-promises/
 - и миллион других ссылок
 - пишем прямо на es6; используем браузер, понимающий es6

2. Создание Promise. Задача 'base'
 - уделить внимание resolve и reject при создании Promise
 - переписать getMessage на getMessageAsync так, чтобы ф-ия возвращала не сообщение, а promise сообщения
 - переписать processElements на processElementsAsync (см комментарии в коде задачи)
 - пусть rejected-сообщение выводится красным цветом
 - запуск base.html должен отрабатывать так же, как раньше + подстветка ошибочных сообщений

3. CSS3 анимация на промисах. Задача 'animation'
 - познакомиться с css3 animations: http://www.w3schools.com/css/css3_animations.asp и др.
 - для решения задачи необходимо обернуть setTimeout в Promise; пусть новая функция называется delay
 - delay возвращает promise, который резолвится (путем вызыва resolve-колбэка) по истечени таймаута
 - выстроить вместо бесконечновложенного утко-кода из setTimeout последовательную цепочку вызовов вида delay().then().then().then()
 - при запуске animation.html анимация на промисах должна отработать в точности так же, как и в исходном виде

4. Работа с API, метод fetch. Задача 'api'
 - познакомиться с fetch: https://davidwalsh.name/fetch, https://learn.javascript.ru/fetch и др.
 - послать три запроса на тестовый api-сервер с помощью fetch:
 - https://jsonplaceholder.typicode.com/posts/1, ...posts/2, ...posts/3
 - обработать их и трижды вызвать соответственно injectIndexedData(index, jsonResult), где index = 1, 2, 3 соответственно
 - при запуске api.html должны отработать три запроса и отобразиться результаты каждого из них

5. Композиция Promise.all. Задача 'api-all'
 - это расширение предыдущей задачи; те же запросы, та же обработка, но
 - нужно вызвать функцию onLoadSuccess после того, как выполнятся все три запроса
 - для этого нам понядобится завернуть наши fetch'и в массив и вызвать Promise.all
 - при запуске api-all.html должны выполнится запросы, отобразиться результаты и показаться строка состояния
 - в тексте задачи описаны три варианта имплементации, нужно попробовать их всех
